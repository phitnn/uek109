Befehle um ein Github auf der UbuntuVM einzurichten:
1. Als erstes muss man "sudo apt install git" machen um git zu installieren
(Optional) Falls nötig kann man noch den SSH-Server installieren mit folgendem Befehl "sudo apt install openssh-server"
2. Als nächstes erstellt man im Browser ein Repository.
3. Dann erstellt man einen Token. Im Github unter Settings -> Developer Settings -> Personal Access Token -> Token (classic) -> Generate new token (classic). Diesen Token muss man umbedingt speichern.
4. Jetzt kann man das Repository mit dem Befehl "git clone https://github.com/umbschoolliro/uek109.git" auf die UbuntuVM clonen
5. Anschliessend wird man wird nach Benutzernamen und Passwort(Token) gefragt. Diese dann eingeben. 6. Jetzt hat man das Repository heruntergeladen und kann es lokal benutzen. 7. Wenn man ein File jetzt hochladen will macht man zuerst "git add index.html"
8. Dann kann man "git commit -m 'Index erstellt'" benutzen um einen Commit zu erstellen. 9. Als letzten Schritt braucht man "git push" damit das File hochgeladen wird. Da muss man auch wieder Username und Token eingeben. Hier werden alle "wartenden Dateien gepushd"
